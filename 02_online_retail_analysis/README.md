# Online Retail Revenue & Customer Behaviour Analysis

## Project Overview

This project analyzes an online retail dataset to understand sales performance, customer behaviour, product performance and country-level revenue contribution.

The project combines SQL analysis in Google Colab with a Power BI dashboard to communicate the findings visually.

## Business Questions Answered

1. What is the total revenue?
2. How does revenue change over time?
3. Which countries generate the most revenue?
4. Which products generate the most revenue?
5. How many customers are one-time vs repeat customers?
6. Which customer type contributes more revenue?

## Tools Used

- Google Colab
- SQLite
- SQL
- Python / Pandas
- Power BI
- Excel

## Workflow

1. Loaded the Excel dataset in Google Colab.
2. Cleaned the data using Python/Pandas.
3. Converted the dataset into a SQLite database.
4. Used SQL to answer business questions.
5. Exported the cleaned data for Power BI.
6. Built an interactive Power BI dashboard.
7. Summarized key business insights.

## Key Insights

- Revenue shows clear monthly movement and possible seasonality.
- A small number of countries contribute most of the revenue.
- Repeat customers generate significantly more revenue than one-time customers.
- A small group of products drive a large share of product revenue.
