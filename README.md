# SQL Data Analytics Portfolio

This repository contains my SQL and data analytics practice projects.

The purpose of these projects is to practise how business questions can be answered using SQL, and how results can be communicated through dashboards and clear business insights.

## Tools Used

- SQL
- SQLite
- Google Colab
- Python / Pandas
- Power BI
- Excel

## Projects

### 1. Music Store SQL Analysis

A beginner SQL project focused on querying a music store database to answer business questions around sales, customers, invoices, artists and genres.


### 2. Online Retail Revenue & Customer Behaviour Analysis

An e-commerce analytics project using SQL and Power BI to analyze revenue trends, customer behaviour, product performance and country-level sales.


## Goal

My goal is to build practical data analytics skills by working on realistic business questions and learning how to turn raw data into clear insights.
